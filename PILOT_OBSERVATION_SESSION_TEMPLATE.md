# Pilot Observation Session Template

## Why this template exists

This template is meant to be filled out during or immediately after a pilot session.

It is designed to keep observations structured across testers so the product team can compare:
- comprehension
- trust reactions
- density/friction points
- personalization value
- return-use likelihood

Use one copy of this template per tester session.
