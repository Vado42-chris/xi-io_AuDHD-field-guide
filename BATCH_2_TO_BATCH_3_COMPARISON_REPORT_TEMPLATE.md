# Batch 2 to Batch 3 Comparison Report Template

## Why this document exists

The repo now has:
- a Batch 2 execution workbook and closeout report template
- a Batch 3 execution workbook and closeout report template
- a broader readiness planning checkpoint template
- a pilot readiness progression ladder

That is enough to run and close the repeated-use and stabilization rounds.

The next practical need is a **single comparison artifact** that sits above the Batch 2 and Batch 3 closeouts and answers:
- what improved from Batch 2 to Batch 3
- what stayed unresolved
- what got worse
- whether the product is genuinely stabilizing
- whether broader readiness planning is actually better justified after Batch 3

This template is that bridge.
