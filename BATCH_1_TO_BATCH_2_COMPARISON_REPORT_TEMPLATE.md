# Batch 1 to Batch 2 Comparison Report Template

## Why this document exists

The repo now has:
- a Batch 1 execution workbook
- a Batch 1 closeout report template
- a Batch 2 execution workbook
- a Batch 2 closeout report template

That is enough to run and close the first two pilot rounds.

The next practical need is a **single comparison artifact** that sits above the two closeout reports and answers:
- what improved from Batch 1 to Batch 2
- what stayed unresolved
- what got worse
- whether the pilot is genuinely maturing the product
- whether the next step should be another pilot batch, targeted fixes, or a broader readiness review

This template is that bridge.
